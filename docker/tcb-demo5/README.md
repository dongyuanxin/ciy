# TCB 云托管 - SSR 场景

最新代码地址：[dongyuanxin/ciy](https://github.com/dongyuanxin/ciy/tree/master/docker/tcb-demo5)

## 接口列表

| 接口地址     | 接口描述                  |
| ------------ | ------------------------- |
| `/a`         | 返回 `files/a.txt` 的内容 |
| `/b`         | 返回 `files/b.txt` 的内容 |
| `/c`         | 返回 `files/c.js` 的内容  |
| `/`          | 返回静态首页的内容        |
| `/api/hello` | 返回 API Server 接口内容  |
